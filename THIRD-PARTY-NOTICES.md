- **[football-data.org](https://www.football-data.org/)** - Primary fixture and match data  
  License/Terms: See their API terms of service  
  Usage: Match fixtures, scores, official IDs, metadata

- **[ESPN Public Scoreboard](https://www.espn.com/)** - Event enrichment (goals, cards, live status)  
  Terms: Public data aggregation for informational purposes

- **[API-Football](https://www.api-football.com/)** - Optional detailed event and statistics enrichment  
  License/Terms: See their API subscription terms  
  Usage: Live match events, venue details, additional statistics

- **[Telegram](https://telegram.org/)** - Stream link discovery from public channels  
  License/Terms: Telegram API Terms of Service  
  Usage: User-session based message retrieval via GramJS

- **[Supabase](https://supabase.com/)** - Stream link persistence
